# Critical Thinking Series

Welcome to the Critical Thinking Series repository! This public repository hosts slides, handouts, demos, and references for educational talks focused on developing critical thinking skills.

## About the Series

The Critical Thinking Series is designed to help participants develop better reasoning skills, recognize cognitive biases, avoid logical fallacies, and make more informed decisions. Each topic is designed to be consumed independently, providing valuable insights that can be applied immediately.

## Repository Structure

- `topics/` - Contains all talk materials organized by topic
- Each topic follows the naming convention: `NN-topic-slug/`
- Each topic contains:
  - `topic.yaml` - Machine-readable metadata
  - `slides.pdf` - Main presentation slides
  - `handout.pdf` - Take-home reference materials
  - `demo/` - Interactive demonstrations and exercises
  - `references.md` - Additional reading and sources
