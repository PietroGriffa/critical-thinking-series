# Critical Thinking Series

Welcome to the Critical Thinking Series repository! This public repository hosts slides, handouts, demos, and references for each talk in our educational series focused on developing critical thinking skills.

## About the Series

The Critical Thinking Series is designed to help participants develop better reasoning skills, recognize cognitive biases, avoid logical fallacies, and make more informed decisions. Each session builds upon previous concepts while introducing new tools and techniques for clear thinking.

## Repository Structure

- `topics/` - Contains all talk materials organized by topic
- Each topic follows the naming convention: `NN-topic-slug/`
- Each topic contains:
  - `topic.yaml` - Machine-readable metadata
  - `slides.pdf` - Main presentation slides
  - `handout.pdf` - Take-home reference materials
  - `demo/` - Interactive demonstrations and exercises
  - `references.md` - Additional reading and sources
