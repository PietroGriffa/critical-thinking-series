## Overview

Learning to identify and avoid common logical fallacies in reasoning and argumentation. This session focuses on formal and informal errors in logic that can undermine sound reasoning.

## Session Content

### What Are Logical Fallacies?

Logical fallacies are errors in reasoning that undermine the logic of an argument. They can be:

- **Formal**: Errors in logical structure
- **Informal**: Errors in content or context

### Common Informal Fallacies

#### Ad Hominem
Attacking the person making the argument rather than the argument itself.

*Example*: "You can't trust John's economic analysis because he's not a good person."

#### Straw Man
Misrepresenting someone's argument to make it easier to attack.

*Example*: "Senator Smith wants to reduce military spending, so he obviously doesn't care about national defense."

#### False Dilemma
Presenting only two options when more exist.

*Example*: "You're either with us or against us."

#### Slippery Slope
Arguing that one event will inevitably lead to a chain of negative consequences.

*Example*: "If we allow students to redo this assignment, next they'll expect to redo every assignment."

#### Appeal to Authority
Citing irrelevant or questionable authority.

*Example*: "This diet must work because a famous actor endorses it."

#### Circular Reasoning
Using the conclusion as evidence for the premise.

*Example*: "The Bible is true because it says so in the Bible."

### Formal Fallacies

#### Affirming the Consequent
If A then B. B is true. Therefore A is true.

#### Denying the Antecedent
If A then B. A is false. Therefore B is false.

### Building Sound Arguments

1. **Clear Premises**: Start with well-defined assumptions
2. **Valid Logic**: Ensure conclusions follow from premises
3. **True Premises**: Base arguments on accurate information
4. **Relevant Evidence**: Use appropriate supporting material
5. **Consider Counterarguments**: Address potential objections

## Interactive Elements

This session includes practical exercises:

- **Fallacy Identification Game**: Spot fallacies in real-world examples
- **Argument Construction**: Build sound arguments on complex topics
- **Debate Analysis**: Evaluate reasoning in formal debates
- **Media Critique**: Identify fallacies in news and advertising

## Key Principles

- **Separate the Argument from the Person**: Focus on reasoning, not personalities
- **Look for Hidden Assumptions**: What premises are taken for granted?
- **Check the Logic**: Do conclusions really follow from premises?
- **Consider Alternative Explanations**: Are there other possibilities?

## Common Pitfalls to Avoid

1. **False Certainty**: Avoid overconfidence in uncertain situations
2. **Oversimplification**: Complex issues rarely have simple solutions
3. **Emotional Reasoning**: Don't let emotions override logic
4. **Confirmation Seeking**: Look for disconfirming evidence
